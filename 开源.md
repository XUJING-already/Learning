## 1.1  什么是开源

- 源码
- 文档，如二次开发文档和用户使用文档
- 开发环境，告诉二次开发者如何搭建和运行代码
- 允许他人贡献代码，而不是仅仅给别人阅读源码的权限
- 问题，用户提问，维护者答复，问题共享（而不是私聊）
- 问题列表和升级计划，记录当前问题，以及何时解决、何时升级

#####       其他配套设施

-  官网
- 文档，可以和官网整合在一起
- 问答社区
- 及时交流社区，即qq群、微信群

### 规范版本号

- ### 一级，重构版本

- ### 二级，重大功能改进

- ### 三级，小升级或者bug修复

### 规范一级目录

- ### src - 源代码

- ### release - 发布结果

- ### test - 测试单元

- ### doc - 文档

- ### example - 示例
